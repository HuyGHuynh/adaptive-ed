# Alzheimer Training - Google Generative AI

This project uses Google's Generative AI to summarize videos.

## Setup

1. Install dependencies:
   ```
   npm install
   ```

2. Add your Google API Key:
   - Open `.env` file
   - Replace `your_api_key_here` with your actual Google API key

## Run

```
npm start
```

Or directly with Node:
```
node index.js
```
